# Warm Studio System

This project uses a warm beige studio system for PINO STUDIO pages.

## Palette

- Page background: `#faf7f0`
- Section alternate: `#f4efe6`
- Surface: `#fffaf3`
- Soft surface: `#eee6da`
- Text primary: `#111110`
- Text secondary: `#5f5a51`
- Text muted: `#8a8175`
- Lines: `rgba(17,17,16,0.10)`
- Primary action: `#111110` fill with white text

## Typography

- Font family: `Inter`, with system sans-serif fallback.
- Header navigation: `font-medium`, uppercase English labels.
- Hero headline: `clamp(44px, 7vw, 96px)`, weight `500`, line-height `0.98`.
- Section headline: `clamp(32px, 4vw, 56px)`, weight `500`, line-height `1.05`.
- Body: `15px-16px`, weight `400`, line-height `1.8`.
- Labels: `10px-12px`, weight `600`, uppercase, letter spacing `0.18em-0.28em`.
- Do not use negative letter spacing.

## Spacing

- Page max width: `1120px`.
- Section padding desktop: `112px 24px`.
- Section padding mobile: `80px 20px`.
- Use generous vertical whitespace and thin `#111110/10` separators.

## Components

- Primary button: dark fill, white text, compact uppercase label.
- Secondary button: transparent beige surface, dark text, subtle border.
- Outlined button: transparent background, `1px` warm dark border, dark text.
- Inputs: warm surface, `1px` subtle border, `0px` radius, `14px` text, `14px 16px` padding.
- Image cards: warm neutral surfaces, sharp corners, soft hover scale, slight brightness lift.

## Language Rules

- Keep `PINO STUDIO` and top navigation labels in English: `HOME`, `FACILITIES`, `PORTFOLIO`, `INSTAGRAM`, `INSIGHTS`.
- Keep descriptive copy, forms, section details, and footer text in refined Korean.

## Page Structure

1. Hero: warm beige page with a softly visible hero image and dark readable text.
2. Studio intro: calm Korean editorial copy with warm image treatment.
3. Portfolio and facilities galleries: masonry or responsive grid with lazy-loaded images.
4. Contact: restrained warm form surfaces and dark call-to-action buttons.
